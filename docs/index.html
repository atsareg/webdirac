Computing power for your applications unleashed

DIRAC project provides a framework and services for building distributed computing systems spanning hundreds of computing centers across the world

Why to choose DIRAC for your scientific research ?

DIRAC was created to provide computing power for a large high energy physics experiment. This successful experience was generalized to suite the needs of researchers from any scientific domain.

Complete solution

Dirac provides a complete solution for distributed computing systems offering workload and data management services with a special emphasis on large scale operations

Batteries included

The code development follows a carefully crafted programming-testing-release-deployment cycle. The DIRAC distribution includes all the necessary packages for running a full set of services.

Open source

DIRAC is a open source project with the code repository and documentation publicly available. It has got controbutions from dozens of developers from all over the world.

Battle proven

DIRAC is actively used by a large number of small and large scientific communities. DIRAC services are provided by several large scale grid infrastructures.

Always evolving

DIRAC development is driven by the needs of scientific communities. Users and developers are working in a close contact and have regular mettings, hackathons, tutorials and other events.

Solid basis

The DIRAC Consortium unites five world-class scientific institutions working together to develop and promote the DIRAC software.
